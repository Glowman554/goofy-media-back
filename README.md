# Goofy Media Backend
This is the backend for Goofy Media.

Currently very WIP.

(More details will follow with time)



### Requirements 

Requires the nodejs packages in the `package.json` file.

Requires a `.env` file with the following variables:
```
DB_USER="..."
DB_PASS="..."
DB_URL="mongodb+srv://..."
```


### How to run
* `npm install`
* `npm start`
